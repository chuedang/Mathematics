# 🐾 Cat RPG Adventure

โปรเจกต์ระบบนักเรียนผจญภัย (Gamification) สำหรับจัดการภารกิจและการแต่งตัวตัวละครแมว

## 🚀 ลิงก์เข้าใช้งาน
[คลิกเพื่อเข้าสู่ระบบ](https://yourusername.github.io/ชื่อ-repository-ของคุณ/)

## 🛠 โครงสร้างระบบ
- **Frontend:** HTML5, CSS3, JavaScript (GitHub Pages)
- **Backend:** Google Apps Script
- **Database:** Google Sheets

## 📖 วิธีการใช้งาน
1. เข้าสู่ระบบด้วยชื่อและรหัสผ่านนักเรียน
2. ทำภารกิจใน Google Sheets เพื่อรับคะแนนและเลเวล
3. ปลดล็อกไอเทมตามระดับเลเวล และกด "บันทึกการแต่งตัว"
